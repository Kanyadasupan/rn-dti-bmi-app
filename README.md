# ⚖️ BMI Calculator App
### Track your Body Mass Index quickly and easily 📊
**Mobile Application developed with React Native Expo**

ยินดีต้อนรับสู่แอปพลิเคชันคำนวณค่าดัชนีมวลกาย (BMI Calculator) ตัวช่วยที่ออกแบบมาเพื่อให้คุณเช็คเกณฑ์น้ำหนักและประเมินสภาวะสุขภาพเบื้องต้นได้อย่างรวดเร็วและแม่นยำ เพียงแค่กรอกน้ำหนักและส่วนสูงของคุณ

---

## ✨ คุณสมบัติหลัก (Features)
* **⚖️ BMI Calculation:** คำนวณค่าดัชนีมวลกาย (BMI) ได้อย่างรวดเร็วและแม่นยำ
* **📊 Health Status Indicator:** แสดงผลลัพธ์พร้อมบอกเกณฑ์สุขภาพที่เข้าใจง่าย (เช่น น้ำหนักต่ำกว่าเกณฑ์, น้ำหนักปกติ, น้ำหนักเกิน ฯลฯ)
* **🎨 Simple & Clean UI:** อินเทอร์เฟซออกแบบมาให้เรียบง่าย ใช้งานสะดวก ไม่ซับซ้อน
* **📱 Responsive Design:** รองรับการทำงานอย่างสมบูรณ์แบบบนสมาร์ทโฟนทุกขนาด ทั้งระบบ iOS และ Android

## 🚀 การติดตั้งและใช้งาน (Setup & Installation)

1.  **Clone โปรเจกต์:**
    ```bash
    git clone [https://github.com/KanyadaSupan/bmi-calculator-app.git](https://github.com/KanyadaSupan/bmi-calculator-app.git)
    cd bmi-calculator-app
    ```
    *(ตรวจสอบและเปลี่ยน URL ของ Repository ให้ตรงกับชื่อ Repository ของคุณ)*

2.  **ติดตั้ง Dependencies:**
    ```bash
    npm install
    ```

3.  **รันแอปพลิเคชัน:**
    ```bash
    npx expo start
    ```
    *จากนั้นสแกน **QR Code** ผ่านแอป **Expo Go** บนมือถือของคุณ (iOS/Android)*
